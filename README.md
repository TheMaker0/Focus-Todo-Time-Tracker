# Focus-Todo-Time-Tracker
Focus is a single-page task and time management dashboard built for people who want their day organized without the clutter. Tasks are sorted into Todo, Ongoing, Pending, Done, and Notes, each with its own color-coded status, and a sidebar keeps live counts so you always know where things stand at a glance.
